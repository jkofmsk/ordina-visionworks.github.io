Data Science with Spark

Spark is a fast and general engine designed for large-scale processing. It runs programs much faster than Hadoop MapReduce and allows multiple programming languages like Java, Scala, Python & R. Being able to use Spark, implies being able to combine the art of data science & big data. Learn the data science with using Spark with this two-day workshop.

Python track

This course combines online learning, hands-on exercises and instructor-led training and will bring you from a beginner level to an advanced level.

Tableau track

Tableau, Power BI & Qlikview are all developed to make analyzing data fast, easy, beautiful and most important, useful. Follow this 2.5 day course and go from zero to hero in Tableau.

MS Data Platform & BI Curriculum - What you'll learn

Go To DataTalks

Advanced Analytics

Advanced Analytics is a huge field, even if we focus on the stack Microsoft provides us. This course takes you from an overview of commonly used tools such as R Server, Azure Machine Learning Services, Azure Data Warehouse and Azure Data Catalog to solutions build using these tools and related best practices.

BI Development

Get to know every aspect of BI Development. Starting off with modeling your data warehouse, filling it up through an ETL process and populating your analysis cube to provide business users with their valued data.

BI Reporting

Start your BI journey here. The course provides participants with an overview of the Microsoft BI stack and using the expertise at Ordina to take participants from the basics in Microsoft BI to advanced solutions build using best practices. Improve your skills while preparing for and MCSA certification.

Big Data Engineering

A big data engineer is continuously challenged with petabytes of data. They understand how to solve big data problems and how to develop innovative solutions. A big data engineer focusses on collecting, parsing, managing, visualizing and analysing big sets of data and turn it into useful information using multiple platforms. They are capable to architect the best solution and develop the necessary prototypes and proof of concepts.

BIML

During development & maintenance phases of your BI environment, you will notice that there are a lot of repeating patterns throughout your solution. These can exist in your database model, your semantic layer, but most of all in your ETL process. With BIML you will implement the DRY (Don't Repeat Yourself) concept throughout your development, making your life easier as repeating architectural changes make their way into the scope. Learn how to harness the power of BIML to speed up your development and maintenance tasks on your ETL.

Cloud Technology

The cloud provides a scalable and growing collection of integrated cloud services. Developers and IT Professionals can use to build, deploy and manage applications through a global network of datacenters. You get to build solutions for proactive and personalized services, reimagine customer experience, deliver differentiated experiences, gain insights and respond quickly to customers and marketing trends.

Data Analytics

Microsoft provides tools for the entire Data Analytics stack. This course introduces you to these tools while making sure you have the necessary knowledge to continue your journey on the path that you like the most.

Data Engineering with Azure

At VisionWorks, we work with more than just SQL Server. Data Engineering with Azure guides you through the different database technologies in Azure. From Azure SQL Database and PostgreSQL to NoSQL solutions like CosmosDb and Azure Tables. Learn what is available and how to use it while preparing for an MCSA certification.

Database Administration

Database administration is an important function in any organization that is dependent on one or more databases. This is usually a dedicated role in the IT department in the large organization. It can be outsourced or contracted to specialized vendors. The primary role of a database administrator includes Database Security, Database Tuning, Database Monitoring, Backup and Recovery.

Database Development

How will your data be stored, how will your data be consolidated, which structure should you database have, what data-types are required, which indexes are needed, which database type is right for the job? All these questions are answered by database developers. Just like regular developers, they concern themselves with the deployment of the databases, their versioning, the continuous integration and automated deployments of the database projects, the prototyping and lifecycle management.

DataVault

In recent years this datawarehouse modeling technique has received a lot of attention in the spotlight. Learn how it differs from the conventional dimensional models and why it matters for large scale data environments. Model your own DataVault and learn the basic patterns for populating it with new data.

GDPR

The General Data Protection Regulation (GDPR) is a regulation by which the European Parliament, the Council of the European Union and the European Commission intend to strengthen and unify data protection for all individuals within the European Union (EU). It also addresses the export of personal data outside the EU. The GDPR aims primarily to give control back to citizens and residents over their personal data and to simplify the regulatory environment for international business by unifying the regulation within the EU.

Master Data Management

Master data management (MDM) comprises the processes, governance, policies, standards and tools that consistently define and manage the critical data of an organization to provide a single point of reference. MDM solutions provide tooling to manage the flow of data change through an organization.

MS Big Data Solutions

How can you implement Big Data solutions using the Microsoft stack? This course is designed to teach you basics about technologies like Hadoop, Hive, Spark, Storm and it's Azure flavours in HDInsight. The course also introduces you to the Lambda Architecture and where each technology fits so that you have all the knowledge you need to start building your own solutions.

Power BI

Power BI is everywhere these days. From on-premises to cloud and hybrid solutions and even in embedded and mobile solutions. As BI developers we do have to keep in mind that Power BI is much more than just a visualization tool. It includes the powerfull SSAS Tabular and Power Query engines which both require an expertise on their own. Learn Power BI end to end and prepare for the MCSA certification.

SQL Server Analysis Services (SSAS)

SSAS provides you with the tools to create an easy-to-understand semantic layer on top of your data warehouse. Your business users can connect to it, analyze their data and form their conclusions in a way that was never before possible on just your SQL data warehouse.

SQL Server Integration Services (SSIS)

To load the data throughout your enterprise into the data warehouse, you need an ETL tool. SSIS is the tool of choice for many because of its simplicity to get started with it, and its extendability if you need to load some unconventional data. Start off your SSIS development and upgrade your skills with best practices and advanced ETL patterns.
